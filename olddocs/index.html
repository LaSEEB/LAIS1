LAIS v0.2beta (20/02/2009)

Download LAIS light (binaries) for users.

Download LAIS complete (bin+src+doc) for more curious users or developers.

Download LAIS full eclipse project for developers using Eclipse.

Download Introduction to LAIS.

  • Changes in spaces: renamed HexTorusSpaceFactory to HexSpaceFactory; added SquareSpaceFactory (with <moore> tag, which if true will provide a Moore neighborhood square space); added <torus> tag to both space factories (if true, space will be toroidal).
  • Added InertialMovementCondition JUNIT test and corrected some bugs.
  • Improved load balancing (threads request cells as needed).
  • Added agent views in cells; views can be used by conditions to quickly get information about what agents are in a cell according to a criteria defined by those same conditions.
  • Improved documentation (quantity and quality).
  • Added QuickProfiler class which allows developers to rapidly pinpoint performance bottlenecks.
  • Corrected bug which would not allow simulations with different x and y dimensions.
  • Updated MatLab charting scripts.

LAIS v0.1.1beta (27/01/2009)

  • Improved cell probing regarding agents, with more detailed information.
  • Improved launcher, only shows really essential errors and info.

LAIS v0.1beta (23/01/2009)

  • Solved bugs in concurrent access to agent and substance collections within CA cells.
  • New concurrent random class (ConcurrentRandom), which provides a unique random number stream for each thread. This solves random lib bug in multicore multithreaded simulations, which manifested essentially in Windows.
  • Introduced unit tests with JUnit! For now only ConcurrentRandom and SubConNeighMovCondition classes have tests.
  • Solved bugs in SubConNeighMovCondition.
  • Improved in-simulation exception handling.
  • Substitute System.out.println's by adequate logging calls, modifiable during runtime (using log4j).
  • Now uses Ant for building the project.
  • Improved cell probing with detailed information about agents and substances.
  • Predator-Prey with chemotaxis example.
  • About button.
  • More examples in documentation.

LAIS alpha version 0.20081222

  • New ant simulation, which doesn't work (not because of LAIS, but because of the expert ant modeller who made the model).
  • Clean substance repository between runs (secondary runs were always slower)
  • True gene cloning, needs more memory but is NEEDED for conditions and actions to maintain individual state (such as the InertialMovementCondition) and suffer individual mutations (tough currently no condition or action directly undergo mutation).
  • More apropriate exception handling, altough certain exceptions are unacessible to treatment by LAIS because they occur on threads started by Repast classes. Will be improved in next release.
  • Print ticks/percent from x to x iterations with percentage (batch mode).
  • Institutional logos: LaSEEB, ISR, IST.
  • Possible to choose directory where to save information.
  • New condition which incorporates several walk conditions (p.e. mix inertial walk with chemotaxic walk).
  • Fixed bug in inertial walk condition.
