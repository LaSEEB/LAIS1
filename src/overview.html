LAIS is a modeling framework for agent-based simulations. LAIS has the following characteristics:
  • 2D explicitly discrete simulation environment.
  • Transparent multithreading.
  • Modular (create simulations by connecting "pieces").
  • Accessible (simulations described in XML).
  • Extensible (simple to create new "pieces").
LAIS is developed on top of the RepastJ agent simulation toolkit and Simple XML serialization and configuration framework, providing an accessible and modular way of creating any kind of agent-based simulations. @author Nuno Fachada
